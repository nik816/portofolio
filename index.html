// Tentang Saya

Lagi belajar bikin web yang rapi & aman

siswa SMK Ma'arif Walisongo Sidowangi yang memiliki ketertarikan di bidang Rekayasa Perangkat Lunak. Ketertarikan tersebut berawal dari rasa penasaran tentang bagaimana sebuah website dibuat dan mampu mengelola data dengan cepat serta aman.

Saat ini saya sedang mempelajari ekosistem JavaScript. Di sisi frontend, saya menggunakan Tailwind CSS untuk membangun tampilan yang responsif dan nyaman digunakan di berbagai perangkat. Selain itu, saya juga mulai mendalami pengembangan backend untuk memahami cara membangun API yang terstruktur dan andal.

Saya lebih mudah belajar melalui praktik langsung. Karena itu, saya selalu mencoba menerapkan setiap materi yang dipelajari ke dalam proyek sederhana. Melalui Git dan GitHub, saya mendokumentasikan proses belajar dan perkembangan proyek yang saya kerjakan agar lebih terorganisir serta menjadi bekal untuk bekerja dalam tim di masa depan.
